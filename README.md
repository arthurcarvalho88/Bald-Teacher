# Bald Teacher
Professor Careca: A Jornada do Cabelo. Ajude o Professor a realizar o seu maior sonho: recuperar o seu cabelo! Um jogo 2D interativo com mecânicas de sobrevivência, coleta de itens e transição de gêneros (Top-Down para Shooter), culminando em uma batalha épica com cutscenes cinematográficas.
